# README: Chesed Shel Emet (True Kindness)

## Introduction
**Chesed Shel Emet** (Hebrew: חסד של אמת) is a real graphic novel project currently in development. It is a powerful historical drama based on true events, exploring themes of faith, morality, and the psychological toll of war.

This repository is dedicated to the **project management, asset organization, and script tracking** for the book. This structural framework is designed to be robust and agnostic of the creative pipeline—whether the final panels are rendered by a human illustrator, a collaborative AI workflow, or a hybrid process, this system ensures that narrative consistency and asset integrity remain intact.

---

## Project Overview
* **Genre:** Historical Drama / Graphic Novel
* **Setting:** The Yom Kippur War (1973) and its aftermath (1974).
* **Scale:** Approximately 110 pages.
* **Core Theme:** The story follows Miklosh, a sensitive soldier drafted into a military burial unit, tasked with the impossible burden of identifying the fallen and providing closure to families amidst the chaos of war.

---

## Directory Structure

To manage a project of this scale, we use the following hierarchy:

* **`01_Narrative_Core/`**: Contains the "Story Bible," including the full Hebrew synopsis and translated summaries.
* **`02_Characters/`**: Detailed profiles for leads like **Miklosh**, **Major Singer**, and **Gadi Ashkenazi**. Includes visual consistency sheets for 1970s IDF uniforms.
* **`03_Scripts/`**: Sequential scripts broken down into three acts:
    1.  *Part A: The Sinai Desert* (October 1973)
    2.  *Part B: True Kindness / Be'eri* (January 1974)
    3.  *Part C: Eternal Rest* (June 1974)
* **`04_Assets/`**: Historical references, including maps of the Sinai front, 1970s military hardware, and the "Goral HaGra" ritual documentation.
* **`05_Production/`**: Storyboards and final renders.
